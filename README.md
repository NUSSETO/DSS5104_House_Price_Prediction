# DSS5104 CA1 — House Price Prediction

This project builds and evaluates linear regression models to predict house prices.

## Project Components

- **`Code/EDA.ipynb`** — Covers data profiling, distributions, and feature relationships.
- **`Code/Main.ipynb`** — Includes preprocessing, model training, cross-validation, and evaluation.
- **`Code/house_dataset.csv`** — Raw housing dataset used for training and evaluation.
- **`Report/`** — Full written report documenting methodology, findings, and conclusions.
- **`results/`** — Output folder containing model metrics, CSVs, and prediction plots.

## Tools Used

- **pandas / numpy** — Data manipulation and numerical computation.
- **scikit-learn** — Linear models (Ridge), cross-validation, and evaluation metrics.
- **xgboost** — Gradient boosted tree model used as a benchmark.
- **matplotlib** — Visualisation of predictions and feature importance.

## Notes

- HTML exports of both notebooks (`EDA.html`, `Main.html`) are included for easy viewing without running code.
- See `requirements.txt` for exact package version constraints.
